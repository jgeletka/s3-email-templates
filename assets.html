PC Users — Prevent Word from compressing your logo

PCs tend to auto-compress images. Use these settings to ensure your logo stays sharp.

Make it the default for all new documents

  1. Go to File → Options → Advanced
  2. Scroll to Image Size and Quality
  3. In the dropdown, select "All New Documents"
  4. Check "Do not compress images in file"
  5. Set Default resolution = High fidelity
  6. Click OK

This applies to anything created from now on.

Make it permanent — update Normal.dotm

  1. Open a blank Word doc
  2. Apply the settings above (with "All New Documents" selected)
  3. Close Word
  4. When prompted, save changes to Normal template

Every new doc inherits this behavior automatically.

Important limitations

  • These settings do not retroactively fix existing documents
  • Each existing file may still have compression baked in — adjust individually as needed
