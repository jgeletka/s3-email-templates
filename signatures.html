Installation Instructions

Outlook · Mac
  1. Copy the HTML above
  2. Outlook → Preferences → Signatures
  3. Click + to create new
  4. Edit → Paste as HTML
  5. Name it and set as default
Outlook · Windows
  1. Copy the HTML above
  2. File → Options → Mail → Signatures → New
  3. Paste HTML into Notepad, save as .htm
  4. Open .htm in Chrome, select all, copy
  5. Paste into the signature editor
Gmail
  1. Copy the HTML above
  2. Settings → See all settings → Signature
  3. Create new signature
  4. In the editor, click the source / HTML icon and paste
